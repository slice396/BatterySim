# BatterySim

Is een thuisbatterij je geld waard? Dat is een vraag waar ik mee liep. En dit is mijn poging om het te testen. Mapje ergens neerzetten en python gebruiken om main.py te starten. Je kan kiezen voor dynamische kosten en voor een vast aan en teruglever tarief. 

Als je een API code aanvraagt via https://transparency.entsoe.eu/myAccount/webApiAccess kan je gebruik maken van de dynamische prijzen. De tool is nog niet zo slim dat hij op basis van prijzen handeld. Hij probeert dus enkel te laden met de zon en te ontladen als de accu inhoud heeft.

Ik heb enkele accu's in de lijst gezet die ik zo kon vinden waaronder de homewizard PIB en de accu van marstek. De uitkomst is een overzicht met terugverdientijd per accu. Doe er je voordeel mee. Ik zal hem nog updaten in de toekomst.

**V1 - Basisversie**
Enkele batterijen toegevoegd, werking via ENTSO wanneer je dynamische tarieven kiest. Nog geen verdere intelligentie in de accu toegevoegd.
